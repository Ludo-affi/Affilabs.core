THE MATH BEHIND IT:

1. Double Filtering:
   Stage 1: Savitzky-Golay (preserves shape, removes high-freq noise)
   Stage 2: Gaussian smoothing (additional noise suppression)

2. Asymmetric Peak Model:
   T(λ) = T_baseline - A·exp(-((λ-λ₀)/σ_left)²)   for λ < λ₀
        = T_baseline - A·exp(-((λ-λ₀)/σ_right)²)  for λ ≥ λ₀
   Where σ_right > σ_left (red side broader)

3. Kalman Filter (3D state space):
   State: [wavelength, FWHM, depth]
   Prediction: x̂ = x_previous (constant model)
   Innovation: y = measurement - prediction
   Update: x = x̂ + K·y (weighted correction)
   Confidence: 1/(1 + ||innovation||/5)

4. Jitter Detection:
   - Check for sign changes in Δλ (high-frequency oscillation)
   - Temporal coherence: 1/(1 + |d²λ/dt²|)
   - Flag if: >2 sign changes AND amplitude <2nm

5. Physics Constraints:
   - Max wavelength jump: 5nm (physically impossible otherwise)
   - FWHM range: 10-100nm (outside = suspicious)
   - Width increases with wavelength: FWHM(λ) ∝ (1 + α·λ)
