# Sample Data Files

This directory contains example SPR measurement data files for testing and demonstration purposes.

## Files

### `sample_single_cycle_spr.txt` (39 KB)
- **Description**: Single-cycle SPR measurement
- **Use Case**: Quick test of data processing pipeline
- **Contents**: One complete association/dissociation cycle
- **Duration**: ~10-15 minutes of measurement data

### `sample_multi_cycle_spr.txt` (373 KB)
- **Description**: Multi-cycle SPR measurement
- **Use Case**: Full experiment validation and testing
- **Contents**: Multiple injection cycles with baseline measurements
- **Duration**: ~60-90 minutes of measurement data

## Usage

These sample files can be used to:
- Test SPR data processing algorithms without hardware
- Validate UI components during development
- Demonstrate data export formats
- Train users on data interpretation

## File Format

Standard Affilabs SPR data format:
- Tab or comma-separated values
- Timestamp, channel data, wavelength information
- Compatible with data viewer and processing tools

## Adding Your Own Samples

To add new sample data:
1. Collect measurement using main application
2. Export data to text file
3. Copy to this directory
4. Update this README with description
