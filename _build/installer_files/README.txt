REQUIRED FILE MISSING: zadig.exe

To complete the installer, download Zadig:
1. Visit https://zadig.akeo.ie/
2. Download the latest version (zadig-2.9.exe or newer)
3. Rename it to: zadig.exe
4. Place it in this folder: installer_files\zadig.exe

Zadig is used to install WinUSB drivers for the Ocean Optics Flame-T spectrometer.
This allows the seabreeze library to communicate with the detector via USB.

Without Zadig, users will need to manually install the WinUSB driver themselves.
