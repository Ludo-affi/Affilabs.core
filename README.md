# Affilabs.core

Desktop software for SimplexSPR, SimplexFlow, and SimplexPro surface plasmon resonance instruments.

## Download

Download the latest installer from the [Releases](https://github.com/Ludo-affi/Affilabs.core/releases) page.

## System Requirements

- Windows 10 or 11 (64-bit)
- USB-A port
- 8 GB RAM minimum
