UI Prototype Rev 1 - Complete Redesign with Modern Grayscale Theme

This massive update introduces a complete UI overhaul with Rev 1 of the new
interface prototype. This represents months of incremental improvements and
establishes the foundation for the v4.0 UI modernization effort.

## UI Prototype Rev 1 (ui_prototype_rev1.py)

### Major Features
- **Sidebar Navigation System**: Resizable left sidebar with 6 tabs (Device Status,
  Graphic Control, Settings, Static, Export, placeholders)
- **Recording Workflow**: Auto-read mode with visual state indicators, file save
  dialog, and red recording state feedback
- **Export System**: Comprehensive 5-section export interface with data selection,
  channel selection, format options, and quick presets
- **Dual-Graph Layout**: Master-detail sensorgram view with resizable 30/70 split
- **Grayscale Design System**: Complete color palette with consistent styling

### Design System
- Grayscale palette (#1D1D1F, #636366, #3A3A3C, #48484A, rgba backgrounds)
- Apple-inspired typography (SF Pro Text, SF Mono)
- Consistent spacing (4/8/12/16px increments)
- Smooth animations and state transitions
- Comprehensive tooltips and accessibility

### Key UI Components
- Recording indicator with live state display (gray viewing / red recording)
- Signal Assessment always-visible card with status badges
- Compact button sizing for better information density
- Channel toggles with consistent color coding
- Export presets (Quick CSV, Analysis Ready, Publication)

### Technical Improvements
- PySide6 (Qt 6.10.0) implementation
- Custom CollapsibleSection widget for accordion sections
- QSplitter for resizable panels (sidebar, graphs)
- QGraphicsDropShadowEffect for depth
- ~6,200 lines of production-ready UI code
