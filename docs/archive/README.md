# Documentation Archive

This directory contains **84 historical documentation files** from the development of Affilabs 0.1.0.

## Purpose

These documents chronicle the development journey, bug fixes, optimizations, and design decisions that led to the stable 0.1.0 release. They are preserved for:

- **Historical Reference**: Understanding how features evolved
- **Debugging Context**: Troubleshooting similar issues in the future
- **Design Rationale**: Learning why certain approaches were chosen
- **Development Lessons**: Avoiding repeating past mistakes

## Categories

### Polarizer System Development (~23 files)
The complete journey of implementing and debugging the polarizer management system, including the critical S/P position swap fix.

### Calibration Optimization (~12 files)
Evolution of the 8-step calibration sequence, integration time optimization, and LED balancing strategies.

### Bug Fixes & Patches (~14 files)
Various fixes for dark noise, saturation, spectral filtering, wavelength handling, and transmission calculations.

### Performance Improvements (~10 files)
Optimization of sensorgram updates, live mode data acquisition, and UI responsiveness.

### Development Notes (~12 files)
Technical analysis documents, code architecture explorations, and processing pipeline designs.

### Status Updates (~5 files)
Historical milestone markers and success confirmations from development sessions.

### Configuration Guides (~8 files)
Superseded by consolidated documentation in root directory.
