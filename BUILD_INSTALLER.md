# Affilabs-Core Installer Build Instructions

## Prerequisites

1. **Download NSIS** (Nullsoft Scriptable Install System):
   - Visit: https://nsis.sourceforge.io/Download
   - Download latest version (3.10 or newer)
   - Install to default location

2. **Download Zadig**:
   - Visit: https://zadig.akeo.ie/
   - Download latest version: `zadig-2.9.exe` or newer
   - Rename to `zadig.exe`

## Directory Structure

Create the following structure:

```
Affilabs-core/
├── installer.nsi              (NSIS script - already created)
├── dist/
│   └── Affilabs-Core.exe     (Generated by PyInstaller)
├── installer_files/
│   └── zadig.exe             (Download from zadig.akeo.ie)
├── ui/img/
│   └── affinite2.ico         (Your application icon)
└── LICENSE                    (Optional - create a LICENSE.txt file)
```

## Build Steps

### Step 1: Prepare Files

1. **Compile the application** (already done):
   ```powershell
   pyinstaller Affilabs-Core.spec --noconfirm
   ```
   This creates `dist\Affilabs-Core.exe`

2. **Create installer_files directory**:
   ```powershell
   mkdir installer_files
   ```

3. **Download Zadig**:
   - Go to https://zadig.akeo.ie/
   - Download `zadig-2.9.exe`
   - Save as `installer_files\zadig.exe`

4. **Create LICENSE file** (optional):
   ```powershell
   echo "Copyright (c) 2025 Affilabs" > LICENSE
   ```
   Or create a proper license file

### Step 2: Build Installer

1. **Open NSIS** (should be in Start Menu after installation)

2. **Compile the script**:
   - Click "Compile NSI scripts" in NSIS
   - Or right-click `installer.nsi` → "Compile NSIS Script"
   - Or use command line:
   ```powershell
   & "C:\Program Files (x86)\NSIS\makensis.exe" installer.nsi
   ```

3. **Output**:
   - Creates `Affilabs-Core-Setup-1.0.2.exe` in the same directory
   - This is your standalone installer!

### Step 3: Test Installer

1. Run `Affilabs-Core-Setup-1.0.2.exe` on a test machine
2. Installer will:
   - Install Affilabs-Core.exe
   - Bundle Zadig.exe
   - Automatically detect USB4000 (if connected)
   - Install WinUSB driver automatically
   - Create desktop and Start Menu shortcuts

## What the Installer Does

### Automatic Mode (USB4000 connected during install):
1. Detects USB4000 spectrometer
2. Automatically installs WinUSB driver via Zadig
3. Shows success message
4. Application is ready to use immediately

### Manual Mode (USB4000 not connected):
1. Asks if user wants to install driver later
2. Zadig.exe remains in installation folder
3. User can run Zadig later after connecting device

### Fallback Mode:
- If automatic detection fails, offers to open Zadig manually
- User can select USB4000 and install WinUSB driver interactively

## Troubleshooting

### "Zadig command failed"
- Ensure Zadig has admin rights
- Some systems may require manual driver installation
- Installer will offer to open Zadig manually

### "Cannot find zadig.exe"
- Make sure `installer_files\zadig.exe` exists
- Download from https://zadig.akeo.ie/
- Must be named exactly `zadig.exe`

### "NSIS compiler not found"
- Install NSIS from https://nsis.sourceforge.io/Download
- Add NSIS to PATH or use full path to makensis.exe

- No Python required
- No dependencies needed
- Includes USB driver installation
- Single file to distribute

```powershell
& "C:\Program Files (x86)\NSIS\makensis.exe" installer.nsi
```

Output: `Affilabs-Core-Setup-1.0.2.exe` ready to distribute!
