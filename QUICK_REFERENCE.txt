# 🚀 ezControl - Quick Reference Card

## Start the Application

### ⭐ Best Method (Recommended)
```powershell
.\run_app.ps1
```
**Does everything automatically:** Clears cache, checks processes, runs clean.

### Alternative Methods
```powershell
# No-cache session mode
.\run_no_cache.ps1

# Manual with -B flag
cd src ; python -B main_simplified.py

# With cache diagnostics
cd src ; python clear_cache.py ; python -B main_simplified.py
```

## Common Tasks

### Made Code Changes?
1. Close application
2. Run: `.\run_app.ps1`
3. Done! ✅

### Having Cache Issues?
```powershell
# Kill any running Python
Get-Process python | Stop-Process -Force

# Clear cache manually
cd src ; python clear_cache.py

# Run fresh
cd .. ; .\run_app.ps1
```

### Check for Running Processes
```powershell
Get-Process python -ErrorAction SilentlyContinue
```

### Clear Cache Manually
```powershell
Get-ChildItem -Path src -Include __pycache__ -Recurse -Force -Directory | Remove-Item -Recurse -Force
```
