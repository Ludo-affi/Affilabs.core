## Conventions

- Each PRD covers one product, feature area, or technology initiative
- Status options: `Planning` → `In Progress` → `Complete`
- Link to relevant `docs/` architecture files where applicable
- Tasks tracked inline per PRD (see Task Tracking section within each doc)

## Task Tracking

Tasks are embedded within each PRD under a `## Tasks` section, organized by phase.
Status markers: `[ ]` = pending, `[~]` = in progress, `[x]` = done.
