# Visual C++ Redistributable

Download the Visual C++ Redistributable from Microsoft.

## Download Link:
https://aka.ms/vs/17/release/vc_redist.x64.exe

## Instructions:
1. Click the link above or visit the URL
2. The file will download automatically (VC_redist.x64.exe, ~14 MB)
3. Save it to this folder
4. Do not rename the file

## What is this?
The Visual C++ Redistributable contains runtime components needed by Python applications built with PyInstaller. It's a standard Microsoft component that many applications require.

## Do I need this?
**Yes, highly recommended.** Without it, the application may fail to start on computers that don't already have it installed.

## File Details:
- **Name**: VC_redist.x64.exe
- **Size**: ~14 MB
- **Version**: Visual C++ 2015-2022 Redistributable (x64)
- **Publisher**: Microsoft Corporation
- **Safe**: Yes, official Microsoft component
